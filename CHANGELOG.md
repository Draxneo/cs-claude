# Changelog — Carnes & Sons Owner App

## v2.1.0 — 2026-06-07
- SELF-UPDATING: versioned service worker (network-first) + auto-reload when a new version ships, so the installed app no longer gets stuck on an old build. Added a refresh button in the header and a 'Force update' button in Settings as a manual escape hatch.
- Settings/permissions hub (Alerts tab): enable Notifications, request persistent storage, step-by-step to set the app's Android battery to Unrestricted (web apps can't change that automatically), app version + force update.
